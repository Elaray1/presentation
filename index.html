Parent ➡️ Child — Props


1. In the parent component, we transfer data from the state to the child.

2. In the child component we accept and output

Parent ⬅️ Child — Callback

1. Create the updateData function in the parent. It has an input parameter value, we assign this parameter to the state of our component using the setState function. Boomerang is ready!

2. Pass the updateData function to the child through props. We launched a boomerang.

3. There is a button in the child element, when you click on it, the function that we passed using props will be called. A new parameter for the text is passed to her, and we want to pass it to the parent. Boomerang comes back.
